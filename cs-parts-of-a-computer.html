The Components of the Modern Computer111

Motherboard

The Motherboard is the main PCB of a computer that allows for communication between various components like the CPU and RAM. Motherboards are classified by the sizes Pico-ATX,Nano-ATX,Mini-ATX,Micro-ATX, and Standard-ATX with Pico-ATX being the smallest and Standard-ATX being the biggest and most common. Motherboards can also have different CPU socket types and versions of PCI expansion slots.

Proccessor

A computer's proccessor performs most general proccesses that include basic math,logic operations,and input/output(Unless a GPU is installed).Proccessors of today almost always have two or more cores running at a number most likely in gigahertz and use varying socket types to connect to the Motherboard such as LGA 775 and LGA 1155

Random Access Memory

Random Access Memory abbreviated as RAM is a key part in any modern computer storing the most used instructions for running programs allowing the system to run much faster as instructions don't have to fetched from a much slower hard drive. The current ram standard is DDR4 with most systems having 4 or 8 Gigabytes of ram which is enough for most refular tasks.

Storage

Today's computers use one or both of two mass storage methods, Solid State Drives and Hard Drives. Hard drives were invented more than 60 years ago but cost less than 10 cents per gigabytes where as a gigabyte of SSD space costs almost two dollars. SSD speeds are around 500mbps which is about 10x as fast as a typical HDD which has rates of around 60mbps.
